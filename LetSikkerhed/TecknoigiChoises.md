## Database
The application shout support multiple database options. We will start with PostgreSQL as our primary database, but we will also support other databases such as MySQL and SQL Server in the future. We will use Entity Framework Core as our ORM to interact with the database.
Update to database sehemas shout be backwards and forward compatible to ensure that we can update the database without breaking existing functionality. We will also use database migrations to manage changes to the database schema over time.
